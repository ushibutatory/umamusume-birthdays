カレンダーの設定方法

※設定後、予定が表示・更新されるまで少し時間がかかる場合があります。

Google Calendar の設定

Web ブラウザから操作する場合

1. [設定] - [カレンダーを追加] - [URL で追加]
2. カレンダーの URL を入力し、[カレンダーを追加]
3. カレンダー名や色、通知などをお好みで変更してください。

Thunderbird の設定

for Windows

1. [カレンダー] - [新しいカレンダー]
2. iCalendar 形式で当カレンダーの URL を指定します。
3. カレンダー名や色、通知などをお好みで変更してください。
